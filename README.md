# Bradford Heating & Cooling Website

This repository hosts a GitHub Pages mirror of the public website at https://bradfordheatcool.com/.

The GitHub Pages URL is:

https://Bradfordheatcool.github.io/

This mirror includes the latest fetched homepage, public static assets, route fallbacks for discovered public pages, and a `404.html` fallback for client-side routing.

To refresh it, re-run the mirror update process and push the generated files to GitHub.
