# Bradford Heating & Cooling Website

This repository hosts a GitHub Pages copy of the public website at https://bradfordheatcool.com/.

The GitHub Pages URL is:

https://Bradfordheatcool.github.io/

The repository includes the homepage, referenced public assets, a GitHub Pages `404.html` fallback for client-side routes, and route-directory fallbacks for the main public pages discovered from the live site.
